## [0.3.0] - 2026-07-14

### 修复与关键
- 系统菜单可切换全局快捷键候选；冲突回退时一点固定，不静默改偏好
- 调用层复制成功不再 toast（轻闪卡片即可）；仅失败/需权限时提示
- 调用层视觉收敛：中性底、更强选中态与序号对比，减少冰蓝洗底干扰扫读
- 去掉侧栏重复「新增」，统一走顶栏新增 / ⌘N
- 预览管理按钮图标化（分享/图钉/编辑/删除，Lucide 线框；悬停看文案）
- 粘贴激活：pid 有效且非自己时不再因显示名/进程名字符串不等丢掉激活（修微信等）
- 使用进行中锁定列表/预览点击，降低连点二次粘贴
- 右键菜单按实际尺寸夹进可视区，贴边不再出屏
- 拔掉显示器/分辨率变化时，可见窗口自动回到可用工作区
- 崩溃重载后若窗口仍可见，自动 focus-search 回到调用态
- WebDAV 配置保存失败时 toast，不再静默抛错
- 隐藏分类落盘失败时不改内存、不假成功关闭弹层
- 窗口禁止外链/新窗与非 file 导航，收紧调用层表面
- 右键置顶改走统一 togglePinAtIndex（含落盘失败回滚）
- 分类重命名落盘失败时回滚提示词与隐藏分类状态
- 渲染进程崩溃自动重载，减少托盘还在窗口假死
- 删除/保存/置顶/导入/撤销在落盘失败时回滚内存状态，避免界面与磁盘不一致
- macOS 唤起时 app.focus(steal)+moveTop，降低 app.hide 后抢不到前台的概率
- 中文输入法组合态：Enter / Esc / 数字键 1-9 不再误触发使用或清搜索
- 仅复制/分享写入剪贴板时取消粘贴后的剪贴板恢复倒计时，避免把刚复制内容抢回
- 托盘退出走 app.quit，并在退出前刷完渲染进程落盘队列，避免粘贴统计半路丢失
- 退出时若剪贴板仍是本次提示词，立即恢复用户原剪贴板（不再等 2.8s 定时器）
- 新一次粘贴开始前先收尾上一次剪贴板恢复，避免倒计时与新内容交错
- 未保存编辑时 ⌘N / 顶栏新增 / 侧栏新增 / 编辑其他条目不再静默冲掉草稿
- 侧栏「新建分类」走统一新增路径，同步锁定失焦隐藏并保护草稿
- 所有落盘（粘贴静默保存、删除、导入、编辑保存）串行化，避免并发写互相覆盖
- 未保存编辑时托盘/二次启动/再次 focus-search 不再静默清空草稿，提示先保存或关闭
- 粘贴成功后约 2.8 秒再恢复原剪贴板；再次粘贴会取消上一次恢复，方便同内容多处 Cmd+V
- 仅空白视为空搜索：空格不再卡住数字键 1-9 / E·P；有字符时 ⌫ 仍改搜索
- 搜索框增加清空按钮（⌘⌫ 仍可用）
- 粘贴成功后静默保存改为串行队列，并立即标记列表脏；快速连用不丢统计，再次唤起会按最新排序
- 唤起时记前台与窗口几何并行准备，缩短显示前等待
- 粘贴成功后异步静默保存使用统计，不挡下一次调用
- 搜索态序号角标支持 ⌘N 宽度，避免挤标题
- Tab / ⇧Tab 在搜索框与列表间闭环（进入当前选中，不总是重置到第一名）
- 搜索结果重绘后滚回顶部，相关度第一名始终在视野内
- 预览收起时分区标题更紧；置顶区左边线提示高频区
- 快捷序号移到标题行，1-9 肌肉记忆更醒目
- PageUp / PageDown 按页跳选，长列表少连按方向键

- 有搜索词时 ⌫ 回改搜索词，不再误删提示词；仅空搜索时 Delete/⌫ 删除（可撤销）

- 预览收起时列表更密（单行摘要、隐藏使用次数），一眼扫更多候选项
