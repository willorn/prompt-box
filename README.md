# PromptBox - 你的提示词管家

[![Electron](https://img.shields.io/badge/Electron-32.2.0-47848F?logo=electron)](https://electronjs.org/)
[![Bun](https://img.shields.io/badge/Bun-1.3.4-000000?logo=bun)](https://bun.sh/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

一款专为效率而生的桌面提示词管理工具。无论你是 AI 创作者、开发者还是内容运营，它都能帮你快速管理和调用常用提示词。

本项目来源于 https://github.com/fantasyao/prompt-master 。

## ✨ 为什么选择 PromptBox？

| 痛点 | 解决方案 |
|------|----------|
| 每次用 AI 都要重复输入相同的提示词？ | `Alt+E` 唤起 → 点选/回车即粘贴 |
| 提示词散落在各处，找不到？ | 多词搜索 + 相关度排序 + 置顶/最近 |
| 多台电脑提示词不同步？ | WebDAV 云同步，配置秒迁移 |
| 窗口遮挡工作界面？ | 用完自动隐藏；失焦退场；全屏应用上方可唤起 |

## 🚀 快速开始

### 下载安装

前往 [Releases](https://github.com/willorn/prompt-box/releases) 页面下载最新版本：

- **macOS**: `PromptBox-x.x.x.dmg`
- **Windows**: `PromptBox-x.x.x.exe` (计划中)
- **Linux**: `PromptBox-x.x.x.AppImage` (计划中)

### 从源码运行

```bash
# 克隆项目
git clone https://github.com/willorn/prompt-box.git
cd prompt-box

# 安装依赖
bun install

# 开发模式
bun run dev

# 打包构建
bun run build

# 运行单元测试
bun test
# 或: npm test

# 发布前质量检查（语法 + 单测 + 主路径符号守卫）
bun run verify
```

## 📖 使用指南

### 基础操作

**使用提示词**
1. 应用启动后默认在后台待命；按 `Alt+E` 呼出窗口（或点击托盘图标）。窗口显示时会置顶，并可在全屏应用上方出现
2. 搜索或方向键选中提示词
3. **单击 / Enter / 1-9（空搜索）/ ⌘1-9**：复制并尽量自动粘贴到刚才的前台应用（成功后约 1 秒恢复你原来的剪贴板）
4. **Shift+单击 / Shift+Enter**：只复制到剪贴板，窗口不隐藏（方便连续取用）
5. 已授权辅助功能时自动粘贴；未授权时仍会复制成功，并提示去授权

### macOS 权限说明

首次使用以下功能时，macOS 可能会要求你授权：

- **自动粘贴**：需要「辅助功能」权限
- **全局快捷键 `Alt+E`**：某些机器上同样可能依赖「辅助功能」权限

建议优先测试打包版 `PromptBox.app`。如果功能未生效，请前往：

`系统设置 -> 隐私与安全性 -> 辅助功能`

确认 `PromptBox` 已被允许控制电脑。

**添加提示词**
1. 点击左下角「新增」
2. 填写名称、选择标签、输入提示词内容
3. 点击保存

**管理提示词**
在任意卡片上右键，或使用右侧预览按钮；也可在非输入态用快捷键：
- **复制内容** - 只复制正文（`⌘/Ctrl+Shift+C`，或 Shift+单击）
- **创建副本** - `⌘/Ctrl+D`
- **分享** - 复制 Markdown / JSON
- **置顶** - `P` 或 `⌘/Ctrl+Shift+P`
- **编辑** - `E`；弹层内 `⌘/Ctrl+Enter` 保存
- **删除** - 空搜索时 `Delete` / `⌫`（可撤销；有搜索词时 ⌫ 改搜索）
- **安全快照** - 内容变更会自动备份；设置菜单可「恢复安全快照」

**导入 / 导出**
1. 点击左下角「系统」→「导出」：可选择 JSON / Markdown / CSV
2. 点击「导入」或直接把文件拖进窗口：支持 JSON、Markdown（按 `#` 标题分条，可选 `标签:` / `tag:`）、CSV
3. 已有数据时可选 **合并导入**（跳过同名同内容）或 **替换全部**
4. 删除或替换导入后，可在 toast 点「撤销」，或按 `⌘/Ctrl+Z`

**登录时启动**
- 托盘图标右键菜单，或「系统」→「登录时启动」
- 开启后会在登录时静默启动，不弹窗抢前台

### 云同步设置

**开启 WebDAV 同步**
1. 点击右上角「⋯」→「同步与备份」
2. 填写 WebDAV 信息（以坚果云为例）：
   - 服务器地址：`https://dav.jianguoyun.com/dav/`
   - 用户名：你的坚果云邮箱
   - 密码：坚果云第三方应用密码
3. 点击「测试连接」→「立即备份」
4. （可选）开启「自动备份」，设置备份频率

**配置迁移到新电脑**
1. 旧电脑：打开「同步与备份」→ 点击「复制配置」
2. 新电脑：打开「同步与备份」→ 点击「粘贴配置」→ 粘贴剪贴板内容 → 点击「应用配置」
3. 点击「从云端恢复」，所有提示词自动下载

## 🎨 核心特性

### 暗色模式
系统菜单可切换「跟随系统 / 浅色 / 深色」，也可用 `⌘/Ctrl+Shift+L`。

### 即点即用
点击卡片后优先自动粘贴到唤起前的前台应用；权限不足时回退为复制成功并提示授权。窗口自动隐藏，失焦也会退场，尽量不打断工作流。

### 优雅的苹果风格
没有繁杂的按钮和菜单，界面干净得像原生 macOS 应用。半透明效果、圆润的边角、流畅的动画，每一次交互都是一种享受。

### 右键快捷操作
在任意提示词卡片上右键，弹出快捷菜单：置顶/编辑/删除。默认界面干净简洁，只有需要管理时才显示操作选项。

### 秒级搜索
支持多词搜索与相关度排序（名称 > 标签 > 内容，并综合使用频率）。默认列表按置顶 / 最近 / 全部组织，越用越顺手。

### 多端同步（WebDAV）
- 支持坚果云、Nextcloud 等任意 WebDAV 服务
- 自动备份：设置一次，按配置频率自动备份到云端
- 配置迁移：复制配置 → 新设备粘贴 → 完成，无需重复填写服务器信息

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+E`（可切换） | 显示/隐藏主窗口；系统菜单可切换候选，冲突回退可一点固定 |
| `/` 或 `⌘/Ctrl+K` / `⌘/Ctrl+F` | 聚焦搜索 |
| `⌘/Ctrl+Backspace` | 清空搜索（搜索框 × 同效） |
| 直接打字 | 非输入态即搜索（空搜索时 E/P 仍可管理） |
| `Tab` / `⇧Tab` | 搜索 ↔ 列表（进入当前选中；卡片上 ⇧Tab 回搜索） |
| `Esc` | 关层 / 收 toast → 清空搜索 → 重置分类 → 隐藏窗口 |
| `↑/↓` | 在卡片间切换选中 |
| `PageUp` / `PageDown` | 按页跳选 |
| `Home` / `End` | 跳到列表首 / 尾 |
| `Enter` | 使用当前选中（搜索框同理；无选中时用可见第一名） |
| `Shift+Enter` / `Shift+单击` | 仅复制，不粘贴、不隐藏 |
| `1-9` | 空搜索时使用第 N 名（搜索中仍用 ⌘/Ctrl+1-9） |
| `⌘/Ctrl + 1-9` | 使用当前可见列表第 N 条 |
| `⌘/Ctrl + Shift + 1-9` | 仅复制第 N 条 |
| `E` / `P` / `Delete`/`⌫` | 编辑 / 置顶 / 删除（空搜索；有搜索词时 ⌫ 改搜索） |
| `⌘/Ctrl+N` | 新增提示词（顶栏「新增」；侧栏收起也可用） |
| `⌘/Ctrl+Z` | 撤销删除或替换导入/恢复 |
| `[` | 收起 / 展开预览区 |
| `]` | 收起 / 展开侧栏 |
| `⌘/Ctrl+Shift+L` | 切换主题 |
| `⌘/Ctrl+,` | 打开系统菜单（顶栏「系统」；侧栏收起时也可用） |

## 🛠️ 技术栈

- **Electron** - 跨平台桌面应用框架
- **Bun** - 高性能 JavaScript 运行时
- **electron-store** - 本地数据持久化
- **webdav** - WebDAV 客户端

## 📁 项目结构

```
prompt-box/

├── preload.cjs             # 预加载桥
├── renderer.js             # 渲染进程（调用 UI）
└── package.json
```
